# NBY Website

Site public Next.js de NBY.

## Variables

Copiez `.env.example` vers `.env.local` en développement. En production,
définissez `NEXT_PUBLIC_API_URL` comme variable de build.

## Coolify

- Build pack : Dockerfile
- Base directory : `/website`
- Dockerfile : `/Dockerfile`
- Port : `3000`
- Domaine conseillé : domaine principal de l'église
